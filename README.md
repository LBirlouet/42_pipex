# pipex

The goal of this project is to recreate the behavior of Unix shell command pipelines. This project emphasizes understanding inter-process communication, process creation, and file descriptor management.

## Allowed functions
- malloc
- free
- open
- read
- close
- perror
- strerror
- access
- dup
- dup2
- execve
- fork
- pipe
- unlink
- wait
- waitpid
- write
- exit
